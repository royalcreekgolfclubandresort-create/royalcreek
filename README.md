# Royal Creek Platform 🏌️

ระบบบริหารจัดการ Royal Creek golf resort ครบวงจร — POS หลายร้าน, คลังสินค้า, HR, แม่บ้าน, แจ้งซ่อม และรายงาน

## สถาปัตยกรรม

- **Frontend:** Static HTML + JavaScript (ไฟล์ในโฟลเดอร์นี้ทั้งหมด)
- **Backend:** Supabase (PostgreSQL + RPC functions) — business logic ทั้งหมดอยู่ใน RPC functions ฝั่งฐานข้อมูล
- **Deploy:** Netlify → https://royalcreek.netlify.app (deploy อัตโนมัติจาก branch main)
- **แจ้งเตือน:** LINE ผ่าน Make.com (สรุปยอดรายวัน 20:00 น.)

## กติกาสำคัญของระบบ (Business Rules)

- **เวลาทำงาน:** เกณฑ์ 9 ชม./วัน (เข้างานครั้งแรกถึงออกงานครั้งสุดท้าย) ต่ำกว่า 9 ชม. แสดงเครื่องหมายเตือนในรายงาน เกิน 9 ชม. นับเป็น OT — ไม่หักพักเที่ยง
- **เงินเดือน:** รายเดือน ÷ 30 ÷ 8 = ค่าแรง/ชม. หรือ รายวัน ÷ 8 — OT คูณ 1 เท่า
- **ประกันสังคม:** หัก 5% สูงสุด 750 บาท เฉพาะงวดเต็มเดือนหรือครึ่งเดือนหลัง
- **งวดจ่าย:** เต็มเดือน หรือแบ่ง 1–15 / 16–สิ้นเดือน (ตั้งค่ารายคน)
- **ลงเวลา:** ต้องอยู่ในรัศมี 100 ม. จากคลับเฮาส์ + ถ่ายรูปยืนยัน
- **สินค้าซ้ำ:** รวมด้วย `core_merge_product_safe(keep, drop)` โดย keep = ID น้อยสุด และต้องตรวจสอบก่อนรวมเสมอ
- **ถาดลูกกอล์ฟ (DR):** ประเภท `self_add_only` — พนักงานเพิ่มสต็อกได้ ลดได้จากการขายเท่านั้น
